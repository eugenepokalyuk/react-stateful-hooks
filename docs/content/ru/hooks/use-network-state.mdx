# useNetworkState

Отслеживает статус подключения браузера (онлайн/офлайн). SSR-безопасен — на
сервере возвращает `{ online: defaultOnline }` (по умолчанию `true`). Поле `since`
— время последней смены статуса или `undefined` до первого перехода

<NetworkStateDemo />

```tsx
import { useNetworkState } from 'react-stateful-hooks';

const { online, since } = useNetworkState();

if (!online) return <Banner>Вы офлайн</Banner>;
```

## Сигнатура

```ts
const { online, since } = useNetworkState(defaultOnline?: boolean); // по умолчанию: true
```

| Поле | Описание |
| --- | --- |
| `online` | `true`, когда браузер сообщает о наличии связи |
| `since` | `Date` последнего перехода онлайн/офлайн или `undefined` |
